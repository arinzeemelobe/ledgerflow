# LedgerFlow - Invoice Management App

## Setup Instructions

### Backend
1. `cd backend`
2. `npm install`
3. Create `.env` with `DATABASE_URL=your_neon_postgres_url`
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Features
- Create, read, update, delete invoices
- Filter by status (paid/pending/overdue/draft)
- Dark/light mode toggle
- Mobile responsive
